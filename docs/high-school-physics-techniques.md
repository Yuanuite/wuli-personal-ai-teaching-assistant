# 高中物理解题技巧速查

本文档保留教师和 Agent 常用的解题策略与二级结论。它是阅读速查，不是可无条件套用的公式表。

自动检索时，以 `.claude/skills/build-physics-simulator/references/secondary-conclusions.json` 中带 `conditions` 和 `forbidden` 的条目为准。使用下列任何结论前，都要核对模型、方向、边界和适用条件；不确定时回到基本定律推导。

## 通用策略

1. **几何优先**：能用切线交点定理、弦长公式、矢量图直接看出的关系，不用坐标法+多行代数展开。
2. **验证至少两重**：量纲检查、边界一致性（极限情况）、趋势合理性、几何自洽，四个维度至少覆盖两个。
3. **量纲排除法**：选择题中直接排除量纲错误的选项，无需完整计算。
4. **守恒律优先**：涉及碰撞、变轨、电磁感应等过程，先写守恒方程（动量/能量/质量），再看具体条件。
5. **逆向思维**：末态→初态反推（如斜抛最高点→逆向视为平抛），降低分析复杂度。

## 运动学

6. **初速为零的匀加速**（等时间间隔）：速度比 1:2:3:…，位移比 1:4:9:…（累计），连续等时段位移比 1:3:5:…。位移差 $\Delta x = aT^2$，$x_m - x_n = (m-n)aT^2$。
7. **中间量关系**：$\bar{v} = v_{t/2} = (v_0+v_t)/2$；$v_{x/2} = \sqrt{(v_0^2+v_t^2)/2}$；$v_{x/2} \geq v_{t/2}$（加速时前者在后）。
8. **0-v-0 模型**：静止→匀加速至 $v_m$→匀减速至零，$a_1/a_2 = t_2/t_1$，全程 $\bar{v} = v_m/2$。
9. **刹车陷阱**：先算刹车时间 $t_0 = v_0/a$，输入时间超过 $t_0$ 时只能用 $v_0^2 = 2ax$ 算位移。
10. **等时圆**：从竖直圆最高点沿任意光滑弦滑到底端时间相等 $t = 2\sqrt{R/g}$；$45°$ 斜面时间最短。
11. **平抛推论**：速度偏角 $\theta$ 与位移偏角 $\alpha$ 满足 $\tan\theta = 2\tan\alpha$；速度反向延长线过水平位移中点。
12. **斜面平抛**：落在斜面上 → 竖直/水平位移 = $\tan\theta$，时间 $t = 2v_0\tan\theta/g$；垂直打在斜面上 → 水平/竖直速度 = $\tan\theta$。
13. **关联速度**：绳/杆端物体对地速度为合速度，沿绳（杆）方向分量相等，垂直方向分量用于确定转动角速度。
14. **竖直上抛对称性**：上升和下降同高度处速度大小相等、方向相反，时间对称，可直接全过程统一列式。

## 力与平衡

15. **$\mu = \tan\theta$ 临界**：斜面上物体 $\mu = \tan\theta$ 时恰能匀速下滑；$\mu > \tan\theta$ 静止；$\mu < \tan\theta$ 加速下滑。在临界模型中施加竖直向下的力，不改变切向与法向的临界比例。
16. **内力公式**：$m_1$、$m_2$ 在力 $F$ 推动下一起加速，$m_1$ 与 $m_2$ 间相互作用力 $F_{内} = \frac{m_{未受力}}{m_{总}} F$。与有无共同摩擦无关，平面/斜面/竖直均可在同加速度条件下使用。
17. **系统牛二律**：$F_{合} = m_1a_1 + m_2a_2 + \cdots$（矢量式），用于连接体问题省去逐体受力分析。
18. **动态平衡—竖小平大**：三力平衡，一力恒定（重力），一力方向不变（水平），另一力方向变化时：竖直方向力变小 → 水平方向力变大。
19. **动态平衡—垂直最小**：三力平衡中第三个力与方向不变的力垂直时，该力取最小值。
20. **晾衣杆模型**：$\sin\theta = d/L$（$d$=杆距，$L$=绳长）。端点上下移 → $\theta$ 不变，拉力不变；杆左右移 → “竖小平大”。
21. **分离临界**：两物体刚好分离 ↔ 接触面弹力为零；还需结合临界时的运动学相容条件。弹簧连接的具体分离位置要由题设约束确定，不一定是弹簧原长位置。
22. **弹力突变**：轻绳、轻杆弹力可突变；弹簧长度不能突变，因而理想弹簧弹力不突变。
23. **摩擦角**：$\tan\phi = \mu$，在滑动或恰达最大静摩擦时，支持力与摩擦力的合力（全反力）与法线夹角为 $\phi$。

## 圆周运动与天体

24. **绳/内轨模型**（竖直圆）：从最高点恰不脱离的临界条件是 $v_{\min}=\sqrt{gR}$；若从最低点恰好能完成整圆，则最低点 $v=\sqrt{5gR}$，最高与最低点拉力差为 $6mg$。
25. **杆/管模型**（竖直圆）：约束能承受推拉时，最高点可取 $v_{\min}=0$；$v=\sqrt{gR}$ 是最高点径向约束力变号的临界。
26. **水平圆运动**（圆锥摆）：$F_{向}=mg\tan\theta=m\omega^2r=mv^2/r$，$a=g\tan\theta$。
27. **高轨低速大周期**：对同一中心天体的圆轨道，轨道越高 → $v$、$\omega$、$a_n$ 越小，$T$ 越大；与卫星质量无关。
28. **黄金代换**：$GM=gR^2$（地表忽略自转），近地卫星 $v_1=\sqrt{gR}\approx7.9\text{ km/s}$；近表圆轨道周期满足 $\rho=3\pi/(GT^2)$。
29. **双星系统**：在相互引力提供向心力的圆轨道模型中，$\omega$ 相同，$m_1r_1=m_2r_2$，$r_1=m_2L/(m_1+m_2)$，$G(m_1+m_2)=4\pi^2L^3/T^2$。
30. **卫星变轨**：低圆轨道转高圆轨道的标准霍曼转移需两次加速；椭圆轨道上 $v_{近}>v_{远}$；开普勒圆轨道机械能 $E=-GMm/(2r)$。
31. **冲日/追及**：相对角位移满足 $(\omega_1-\omega_2)t=2n\pi$；具体题干若要求相对或相反方向，再按初末相位修正右端。

## 动量与能量

32. **一维弹性碰撞—动碰静**：$v_1'=\frac{m_1-m_2}{m_1+m_2}v_1$，$v_2'=\frac{2m_1}{m_1+m_2}v_1$。推论：$m_1=m_2$ 速度交换；$m_1\gg m_2$ 时小球约以 $2v_1$ 弹出；$m_1\ll m_2$ 时小球近似原速反弹。
33. **相对速度法**（一维弹性碰撞）：碰前接近相对速度等于碰后分离相对速度；再与动量守恒联立。
34. **人船模型**：水平方向外力冲量为零且初总动量为零时，$m_1s_1=m_2s_2$，再结合 $s_1+s_2=L_{相对}$。
35. **折合质量**：二体质心系中 $\mu=m_1m_2/(m_1+m_2)$，相对运动动能为 $E_k=\frac12\mu v_{相对}^2$。
36. **功能图像规律**：$F-x$ 图有向面积=功；一维保守力系统中 $E_p-x$ 图斜率 $=-F$；$E_k$ 极值点需结合 $F=0$ 及两侧变号判断极大或极小。
37. **流体冲击**：稳定流束在时间 $\Delta t$ 内质量为 $\rho Sv\Delta t$；沿入射方向停止时 $F=\rho Sv^2$，原速反弹时 $F=2\rho Sv^2$。

## 静电场

38. **三个点电荷平衡**：三电荷共线且仅受库仑力时，两侧同号、中间异号，并满足“两同夹一异，两大夹一小”。
39. **平行板电容器动态分析**：忽略边缘效应时，接理想电源（$U$ 不变）：$d\uparrow\Rightarrow C\downarrow\Rightarrow Q\downarrow\Rightarrow E\downarrow$；断开后电荷量保持（$Q$ 不变）：$d\uparrow\Rightarrow C\downarrow\Rightarrow U\uparrow$，$E$ 不变。
40. **带电粒子在匀强电场中偏转**（初速与电场垂直）：$y=qUL^2/(2mdv_0^2)$；类平抛轨迹的末速度反向延长线过水平位移中点。
41. **等效重力场**：对匀强电场与重力场，先作矢量和 $\vec g_{\mathrm{eff}}=\vec g+q\vec E/m$；只有两者垂直时，大小才是 $\sqrt{g^2+(qE/m)^2}$。
42. **已知三点电势求场强**：在匀强电场中用等分法找等势点，连成等势线，电场与其垂直，再用 $E=U/d$。

## 恒定电流

43. **串反并同**：在常见闭合电路动态分析中，某电阻增大时，与之串联支路的相关量往往反向变化，并联支路往往同向变化；使用前要核对电源内阻、仪表和支路拓扑。
44. **电源最大输出功率**：纯电阻外电路中，$R_{外}=r$ 时 $P_{\max}=E^2/(4r)$。
45. **电表内外接选择**：“大内大，小外小”——待测电阻大用电流表内接法，小用外接法；用相对误差判断时的常用临界值为 $R_0=\sqrt{R_AR_V}$。
46. **理想电表和稳态含容电路**：理想电流表→导线，理想电压表→断路，直流稳态后电容器支路→断路。

## 磁场

47. **带电粒子在匀强磁场中的圆周运动**：$|q|vB=mv^2/r$，所以 $r=mv/(|q|B)$、$T=2\pi m/(|q|B)$。再用弦切角、偏向角和弦长确定圆心与实际圆弧。
48. **有界磁场对称性**：对直线边界的同一段圆弧，射入与射出相对边界的角相等；圆形磁场的“径向射入、径向射出”要同时核对轨道圆与磁场圆的几何条件。
49. **磁聚焦/磁发散**：当轨迹圆半径 $r$ 等于圆形磁场半径 $R$ 时，可利用等半径几何得到同点不同方向射入与平行射出、或平行射入与同点汇聚的对偶关系。
50. **速度选择器**：电场力与磁场力方向相反时，$v=E/B$ 的粒子直线通过，与电荷量和质量无关。
51. **安培力等效**：闭合线圈在匀强磁场中所受安培力合力为零；一段弯曲导线的合力等效于连接两端点的直导线所受力。
52. **配速法**：带电粒子在某些均匀复合场中做摆线运动时，可将速度分解为匀速分量和圆周分量；使用前必须证明分解后的方程确实解耦。

## 电磁感应

53. **楞次定律**：“增反减同”判断感应磁场，“来拒去留”判断相对运动趋势；最后再用右手螺旋定则确定电流方向。
54. **感应电荷量**：总电阻不变且电路结构明确时，$q=n|\Delta\Phi|/R_{总}$，与变化经历的时间无关。
55. **转杆发电**：均匀磁场中长为 $L$ 的导体杆绕一端以角速度 $\omega$ 转动时，两端感应电动势大小 $E=\frac12BL^2\omega$。
56. **方均根法求有效值**：正弦交流电 $I_{有效}=I_m/\sqrt2$，$U_{有效}=U_m/\sqrt2$；非正弦波按等效热效应定义分段计算。
57. **变压器等效电阻**：对理想变压器，副边负载等效到原边为 $R_{等效}=(n_1/n_2)^2R_{副}$。
58. **远距离输电**：在输送功率给定的模型中，升压使电流减小，因而降低 $\Delta P=I^2R_{线}$ 和 $\Delta U=IR_{线}$。

## 热学、光学、近代物理

59. **理想气体图像**：$p-V$ 图中等温线为双曲线，过程线下有向面积为气体对外做功；使用 $V-T$ 和 $p-T$ 过原点结论时，温度必须用热力学温度。
60. **分子力与分子势能**：$r=r_0$ 时分子合力为零、分子势能取极小值；势能零点是人为选择，不能由极小直接推断其数值必为负。
61. **全反射临界角**：光从光密介质射向光疏介质时，$\sin C=n_{疏}/n_{密}$；入射角不小于 $C$ 时才发生全反射。
62. **氢原子能级跃迁**：$h\nu=E_m-E_n$，从能级 $n$ 电离需入射粒子或光子提供符合相应机制的能量；一群原子从能级 $n$ 逐级可能跃迁时最多产生 $C_n^2$ 条谱线。
63. **光电效应**：$E_{k\max}=h\nu-W_0$，截止频率 $\nu_0=W_0/h$。频率决定能否发生以及最大初动能；频率和材料不变时，光强主要影响饱和光电流。
64. **三种射线**：$\alpha$ 射线是氦核流，贯穿最弱、电离最强；$\beta$ 射线是高速电子或正电子流；$\gamma$ 射线是高能光子，贯穿最强、电离最弱。
