# 悟理学习站

这是可独立发布到 GitHub Pages 的只读学生端。公开题目必须从本地教师工作台生成预览并由教师确认；不要手工复制 `student-error-library/` 或 `output/` 整个目录。公开目录只包含目录索引、学生版 Markdown、脱敏题图与答案图片、可用时生成的 PDF，以及教师已批准的仿真。

仿真在桌面端采用画布与控制栏并排布局；手机端首屏保留画布、当前结论、播放和进度，次要控制默认折叠。发布前应分别用桌面和手机视口检查。公开 PDF 因字体或图片工具链缺失而跳过时，Markdown 页面仍可发布，页面不应显示无效下载入口。

发布时通过 GitHub Actions 自动部署到 `gh-pages` 分支，GitHub Pages 源设为该分支。
